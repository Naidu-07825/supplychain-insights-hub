## â“ Frequently Asked Questions

### Q: Can I delete my preferred address?
**A:** Yes, but a new address will be automatically set as preferred.

### Q: Can I change address after placing order?
**A:** No, address is locked when order is placed. Create reorder if you need different address.

### Q: What if I delete all addresses?
**A:** You can still place orders using custom address field.

### Q: Does reorder use the same price?
**A:** No, it uses current prices. Better to reorder if prices have decreased!

### Q: Can I search by order ID?
**A:** Not yet, but you can filter by status to narrow down results.

### Q: What does profile completion percentage mean?
**A:** It shows how many fields you've completed (name, email, phone, address).

### Q: Can I have same address with different labels?
**A:** Yes! You can save same address as both "Home" and "Work" if needed.
